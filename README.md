# CRM Activity Dashboard

Streamlit dashboard for tracking activity in VendTech's CRM system.

## Development

This project uses:
- **BMAD-METHOD** for structured AI-driven development
- **Beads** for persistent issue tracking

See `AGENTS.md` for agent instructions.

## Setup
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run
```bash
streamlit run app.py
```
